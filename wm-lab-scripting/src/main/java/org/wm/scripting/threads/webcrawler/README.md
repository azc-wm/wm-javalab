# Webcrawler

Key Topics:
- Thread, Runnable, Callable, Future
- Thread lifecycle, daemon threads, interruptions
- Synchronization (synchronized, volatile, wait/notify)
- Race conditions and visibility
- Thread coordination via join() and interruption

Objectives:

- Build a basic web crawler using manual threads and synchronization.
- Introduce a race condition intentionally, detect it using logging and VisualVM.
- Replace explicit threads with ExecutorService + Callable.
